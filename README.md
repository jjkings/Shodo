# ZenShodo - デジタル書道アプリ

Webブラウザ上で動作する、リアルな書き心地を追求したデジタル書道アプリケーションです。
筆の入り、止め、払いなどの筆圧（速度変化によるシミュレーション）や、和紙の質感を再現しています。

## ✨ 特徴

*   **リアルな墨の表現**: マウスやタッチの速度に応じて筆の太さが変化し、「溜め」や「払い」を表現できます。
*   **3種類のツール**:
    *   **太筆**: 力強い文字を書くためのメインブラシ。
    *   **細筆**: 名前や繊細な線を書くための小筆。
    *   **篆刻（てんこく）**: 作品の仕上げに押す赤い印鑑（ハンコ）。
*   **カスタマイズ**: 設定画面から印鑑の文字や作家名を変更可能。
*   **共有機能**: 書いた作品を画像として書き出し、X (旧Twitter) への投稿シミュレーションが可能です。

## 🚀 起動準備 (セットアップ)

このプロジェクトは React と TypeScript で記述されています。ローカル環境で実行するには、Node.js が必要です。

### 1. 前提条件
*   Node.js (v18以上推奨)
*   npm または yarn

### 2. 開発環境のセットアップ

このソースコードは Vite などのモダンなフロントエンドビルドツール上で動作することを想定しています。

一般的な起動手順（Viteを使用する場合）:

1.  ターミナルを開き、依存関係をインストールします。
    ```bash
    npm install
    ```

2.  開発サーバーを起動します。
    ```bash
    npm run dev
    ```

3.  表示されたURL（例: `http://localhost:5173`）をブラウザで開きます。

## 🎨 使い方

アプリを起動すると、白い半紙が表示されます。

### ツールバーの操作 (画面下部)

左から順に以下の機能があります：

1.  **太筆 (Futo Fude)**
    *   デフォルトのブラシです。大きくのびのびとした文字を書くのに適しています。
    *   *コツ*: ゆっくり動かすとインクが滲んで太くなり、速く動かすとかすれて細くなります（「はらい」の表現）。

2.  **細筆 (Hoso Fude)**
    *   細い線を描画します。署名（雅号）を書くのに最適です。

3.  **篆刻 (Stamp)**
    *   「印」ツールを選択し、キャンバス上の好きな場所をクリック（タップ）すると、朱色の印が押されます。
    *   ドラッグ操作ではなく、スタンプのように一度だけタップしてください。

4.  **消去 (Trash Icon)**
    *   半紙を新しいものに取り替えます（描画内容を全消去）。

5.  **設定 (Settings Icon)**
    *   **篆刻印**: 印鑑の文字（1〜4文字）を変更できます。ここで設定した文字がスタンプに反映されます。
    *   **作家名**: 投稿時に使用する名前を設定します。

6.  **投稿 (Send Icon)**
    *   作品のプレビューを確認し、X (Twitter) へのシェア画面を開きます。
    *   ※現在はデモモードのため、実際の投稿は行われません。

## 📱 推奨環境

*   **デバイス**: PC、タブレット、スマートフォン対応。
*   **入力**: マウスでも操作可能ですが、iPadやペンタブレットを使用するとより直感的な書き心地を楽しめます。
*   **ブラウザ**: Google Chrome, Safari, Edge, Firefox 最新版。

## 🛠 技術スタック

*   **Frontend**: React 19, TypeScript
*   **Styling**: Tailwind CSS
*   **Canvas**: HTML5 Canvas API (独自ブラシエンジン)
*   **Icons**: Lucide React
*   **Font**: Noto Serif JP (Google Fonts)
